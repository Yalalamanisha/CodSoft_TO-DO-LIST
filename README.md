To-Do List Application-Python CLI

Description

A simple command-line To-Do List application built using Python that allows users to manage daily tasks efficiently.
Users can add, view, and delete tasks, with all data stored in a local file to ensure persistence even after the program is closed.

 Technologies Used

| Category        | Tools / Concepts              |
|----------------|------------------------------|
| Programming     | Python                       |
| File Handling   | Python file operations       |
| Concepts Used   | Functions, loops, lists, conditions |
| Version Control | Git & GitHub                |

# Features

* Add new tasks to the to-do list
* View all saved tasks with numbering
* Delete tasks by selecting task number
* Store tasks permanently in a text file
* Simple and interactive command-line interface

# How to Run

1️⃣ Clone the repository
```bash
git clone https://github.com/Yalalamanisha/CodSoft_TO-DO-LIST.git
```

2️⃣ Navigate to the project folder
```bash
cd CodSoft_TO-DO-LIST
```

3️⃣ Run the program
```bash
python todolist.py
```
